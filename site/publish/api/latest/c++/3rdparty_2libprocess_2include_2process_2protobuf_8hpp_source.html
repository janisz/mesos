56 // The rest of this file provides libprocess "support" for using
57 // protocol buffers. In particular, this file defines a subclass of
58 // Process (ProtobufProcess) that allows you to install protocol
59 // buffer handlers in addition to normal message and HTTP
60 // handlers. Install handlers can optionally take the sender's UPID
61 // as their first argument.
62 // Note that this header file assumes you will be linking
63 // against BOTH libprotobuf and libglog.

830  promise.discard();
