35 // This isolator implements support for Container Network Interface (CNI)
36 // specification <https://github.com/appc/cni/blob/master/SPEC.md> . It
37 // provides network isolation to containers by creating a network namespace
38 // for each container, and then adding the container to the CNI network
39 // specified in the NetworkInfo for the container. It adds the container to
40 // a CNI network by using CNI plugins specified by the operator for the
