34 // The provisioner rootfs directory is as follows:
35 // <work_dir> ('--work_dir' flag)
36 // |-- provisioner
37 // |-- containers
38 // |-- <container_id>
39 // |-- backends
40 // |-- <backend> (copy, bind, etc.)
41 // |-- rootfses
42 // |-- <rootfs_id> (the rootfs)
43 //
44 // There can be multiple backends due to the change of backend flags.
45 // Under each backend a rootfs is identified by the 'rootfs_id' which
