114  // Performs 'docker run IMAGE'. Returns the exit status of the
115  // container. Note that currently the exit status may correspond
116  // to the exit code from a failure of the docker client or daemon
117  // rather than the container. Docker >= 1.10 [1] uses the following
118  // exit statuses inherited from 'chroot':
119  // 125 if the error is with Docker daemon itself.
120  // 126 if the contained command cannot be invoked.
