Try<Nothing> systemd::mesos::extendLifetime ( pid_t  child)

A hook that is executed in the parent process.

It migrates the pid of the child process into a the MESOS_EXECUTORS_SLICE in order to extend its life beyond that of the agent.

Returns
Nothing if successful, otherwise Error. Note that returning Error implies the child process will be killed.
