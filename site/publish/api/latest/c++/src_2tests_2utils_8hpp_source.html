30 // Path finding utilities.
31 //
32 // Various tests need to access paths to load files or libraries. Normally,
33 // these paths are located within the build directory, often in the .libs
34 // directories generated by Automake. However, if the Mesos tests were built
35 // with --enable-tests-install, we may be running the tests from an installed
36 // version of the test suite. In this case, we should always find paths in the
37 // installation tree and never the build directory. If the build directory is
38 // still present, you can force these helpers to ignore it by passing the
39 // --build_dir=/none option to the test suite.
40 //
41 // The path helpers below all check for a build directory path, then fall back
42 // to an installation path only if --enable-tests-install was specified at
43 // configuration time.
